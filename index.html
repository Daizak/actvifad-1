Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

1. Web 2.0

Web 2.0: La Web 2.0 se refiere a una evolución de la web que permite la interacción y colaboración de los usuarios. A diferencia de la Web 1.0, que era estática, la Web 2.0 fomenta la creación de contenido por parte de los usuarios a través de plataformas como blogs y redes sociales.

2. Web 3.0 3

Web 3.0: La Web 3.0, también conocida como la web semántica, busca proporcionar un significado más profundo a los datos en línea, facilitando la conexión y el análisis de información de manera más inteligente

3. La W3C y los estándares

La World Wide Web Consortium (W3C) es una comunidad internacional que desarrolla estándares para asegurar el crecimiento a largo plazo de la web. Su objetivo es promover la interoperabilidad y la accesibilidad

Tema 2: El Diseño De Una Página Web

I. Tipos de páginas Web

Páginas informativas: Estas páginas proporcionan información sobre un tema específico, como blogs o sitios de noticias. Páginas de comercio electrónico: Diseñadas para vender productos o servicios en línea, estas páginas incluyen catálogos, carritos de compra y opciones de pago. Páginas de portafolio: Utilizadas por profesionales creativos para mostrar su trabajo, como fotógrafos, diseñadores y artistas. Páginas de aterrizaje: Creadas para convertir visitantes en leads o clientes, a menudo enfocándose en una sola oferta o producto.

II. Puntos básicos en la elaboración de una página Web.

Definir el objetivo: Antes de comenzar, es crucial tener claro qué se quiere lograr con la página. Conocer a la audiencia: Entender quiénes son los usuarios potenciales ayudará a diseñar una experiencia más efectiva. Estructura y navegación: La disposición de los elementos debe ser clara y lógica para facilitar la navegación . Contenido de calidad: El contenido debe ser relevante, conciso y atractivo para mantener el interés del visitante. Diseño responsivo: Asegurarse de que la página se vea bien en dispositivos móviles y de escritorio es esencial en la actualidad .
